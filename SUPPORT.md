# Support

Cue is a small, solo-maintained open-source project. Here's how to get help and
what to expect.

## Where to go

- **Questions, "how do I…", and ideas** → [GitHub Discussions](https://github.com/avonbereghy/cue/discussions).
- **Bugs / something's broken** → [open a bug report](https://github.com/avonbereghy/cue/issues/new?template=bug_report.yml).
- **Feature ideas** → [open a feature request](https://github.com/avonbereghy/cue/issues/new?template=feature_request.md).
- **Security issues** → see [SECURITY.md](SECURITY.md). Please report privately, not as a public issue.

## What to expect

This is maintained in spare time, best-effort. You can usually expect a first
response within **about a week**. If you haven't heard back by then, a polite
ping on the thread is welcome.

## Scope

Cue **monitors** Claude Code sessions — state, tokens, permissions — and reads
local data only. Things outside that scope (changing Claude Code's own behavior,
or anything requiring a backend or cloud service) are generally out of scope
today. See the [Roadmap](README.md#roadmap) for where it's headed.
